# anime_hand_vfx
Real-time hand tracking project that overlays Naruto-style effects (Chidori and Rasenshuriken) using OpenCV and MediaPipe.

Run:
pip install -r requirements.txt
python main.py
Press `ESC` to exit.

Controls:
* Open hand → charge power
* Close hand → release
* Right hand → Chidori
* Left hand → Rasenshuriken

Tech:
Python, OpenCV, MediaPipe, NumPy
